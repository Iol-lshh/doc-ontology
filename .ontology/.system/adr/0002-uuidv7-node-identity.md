# ADR 0002: 노드 식별자는 UUIDv7

- Status: Accepted
- Date: 2026-06-19

## Context

노드 간 관계의 앵커가 필요하다. 파일명(basename)을 앵커로 쓰면 파일을 옮기거나 이름을 바꿀 때 모든 참조가 끊긴다.

## Decision

모든 노드에 **UUIDv7**을 `id`로 부여한다. 관계(graphIndex)는 id로만 참조한다. UUIDv7의 시간순 정렬 특성은 부수적이며, 핵심은 이동·개명에 불변인 안정 앵커다.

## Consequences

- 파일을 옮기거나 이름을 바꿔도 관계가 보존된다.
- id는 사람이 읽기 어렵다 → 사람은 label로 작성하고 빌드가 변환한다([[0006]]).
- id 발급·중복 검사를 빌드가 자동화한다.
