# 왜 본체와 부속을 나누는가

최종 산출물에 근거·과정·이력이 섞이면 verbose해진다. 근거를 지우면 추적성을 잃고,
남기면 인용 출처로서의 단정성을 잃는다. 두 요구가 충돌한다.

본체는 결론만 남겨 깨끗한 인용 출처로 두고, 근거는 부속 노드로 옮긴다.
근거는 사라지지 않고 그래프로 추적되며, 본체는 단정성을 지킨다.
